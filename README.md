# ScoutVision
ScoutVision is a data analysis project for Scout Group Grombelia Medina to transform raw data into actionable insights through a clear and interactive dashboard. The goal of this project is to:

Clean and organize large datasets
Analyze key metrics (KPIs)
Visualize insights in a user-friendly dashboard
This project is ideal for tracking performance, identifying trends, and supporting data-driven decision-making.


# Dataset Used
Multi-source scout data 
Structured into a Data Warehouse (DW) schema
# Features
  # Data Cleaning & Preparation
Handling missing values
Data normalization and standardization
Building a clean star/snowflake schema
Creating relationships between fact and dimension tables
 # Interactive Dashboards
Dynamic and role-based dashboards
Advanced filtering (date, unit, activity type, location)
Clear and modern visualizations
Cross-filtering and drill-down capabilities
   # Advanced Analytics & Machine Learning
Predictive analysis 
Identification of trends and anomalies
AI visuals 
Correlation analysis using scatter charts
  # Key Metrics (KPIs) by Decision Maker
   Chief Scout Group Leader

   Treasurer

   Unit Leaders

# Data Warehouse Schema
Star/Snowflake schema implemented
Fact tables:
Fact Activity
Fact Budget
Dimension tables:
Date
Location
Activity Type
Unit
Members

✔️ Optimized for performance and scalability
✔️ No redundancy, clear relationships
